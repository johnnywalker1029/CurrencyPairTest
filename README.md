# CurrencyPairTest

The project is seperated, includes Java Spring Boot as server side and Vuejs as client side.
Java Spring Boot - Version: 2.3.1.RELEASE

Rest API WebService - Post Trade and Get Posted Trade

Vuejs - Version 2.0

Nodejs client side GUI library for post and load trades

The projects' compilation sequences:
1. Compile Nodejs client to Java Project's /resources/WEB-INF/
2. Compile Java Spring Boot as an bootable jar

The project avaiable to run in locally with the proxy settings between Java Project and Nodjs Client Side.

The local URL for testing link as below:

Post Message GUI: http://localhost:8000/post
Load Message GUI: http://localhost:8000/load
